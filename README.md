# HR Analytics Dashboard (Power BI)

This project analyzes HR data to understand employee attrition, workforce distribution, and key HR metrics using Power BI.

## Project Overview

The dashboard helps HR teams identify patterns related to employee turnover, department performance, and workforce demographics.

## Dataset

The dataset contains employee information such as:

* Employee ID
* Department
* Job Role
* Salary
* Attrition
* Years at Company
* Education
* Age

## Tools Used

* Power BI
* CSV Dataset
* Data Visualization

## Key Insights

* Attrition analysis across departments
* Workforce distribution by job roles
* Employee demographics analysis
* Performance and workforce trends

## How to Use

1. Download the repository
2. Open `HR_Analysis.pbix` using Power BI Desktop
3. Explore the interactive dashboard
